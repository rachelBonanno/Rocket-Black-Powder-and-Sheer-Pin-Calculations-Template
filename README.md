# Rocket Black Powder and Sheer Pin Calculations Template
Excel document that takes your rocket stats and launch location and calculates the amount of black powder and number of sheer pins needed. The sheet is set up for a duel deploy rocket that separates at the nose cone and aft airframe where the drogue is in the aft and main in the forward airframe.

## About the Sheet
When using the sheet you only need to input values into the blue boxes (all labled replace). By filling out these blue boxes out you will generate the values that will be displayed in the gray boxes. If you click on the gray boxes you will find the forumals linked to the blue boxes written out in plain english.
